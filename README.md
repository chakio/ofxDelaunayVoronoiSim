ofxDelaunayVoronoiSim
====

## Description
ドロネー三角分割とボロノイ図の描画  
ドロネー三角分割：散らばった点をポリゴン化された図形  
ボロノイ図；散らばった点の位置に基づき領域分けされた図
双対の関係にあるドロネー三角分割とボロノイ図を同時に描画してみました  

## Environment
* Ubuntu16.04
* openFrameworks v0.10.0
* visualstudio code
